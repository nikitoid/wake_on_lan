# Wake-on-LAN Vercel App

Простое веб-приложение для отправки "Magic Packets" через Vercel Serverless Functions.

## Установка и Деплой

### 1. Локальный запуск (для тестирования)

Для запуска локально вам понадобится `vercel-cli`.

```bash
npm install -g vercel
vercel dev
```
Не забудьте создать файл `.env` в корне с ключом:
```
AUTH_KEY=your_secret_password
```

### 2. Деплой на Vercel

1. Скопируйте файлы в свой репозиторий (GitHub/GitLab/BitBucket).
2. Импортируйте проект в Vercel.
3. **Важно:** В настройках проекта (Settings -> Environment Variables) добавьте переменную:
   - **Name:** `AUTH_KEY`
   - **Value:** (Ваш секретный пароль, который вы будете вводить в форму)

### Структура проекта

- `index.html`: Фронтенд интерфейс.
- `/api/wake.js`: Serverless функция, выполняющая отправку UDP пакета.
- `vercel.json`: Конфигурация маршрутизации.
- `package.json`: Описание проекта.

## Использование

1. Откройте развернутый сайт.
2. Введите IP адрес целевого компьютера (должен быть публичным IP, если вы отправляете через интернет, и на роутере должен быть проброшен UDP порт).
3. Введите MAC адрес целевого сетевого интерфейса.
4. Введите Secret Key, который вы задали в переменных окружения Vercel.
5. Нажмите "Send Magic Packet".
